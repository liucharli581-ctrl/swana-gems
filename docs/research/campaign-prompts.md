# Swana Gems Campaign Prompts

使用方式：
1. 用 Image Prompt 生模特佩戴图（Midjourney / ChatGPT Image / 即梦）
2. 用模特图 + Video Prompt 做图生视频（Seedance / Kling）
3. 4-5 段视频拼成 12-15s banner
4. 剪映加文字

## Banner 视频剪辑指南

### 首页 Hero Banner (15s)
```
镜头1 (3s): Elisa Pendant 模特窗边氛围 → 字幕 "Summer 2026"
镜头2 (4s): Crystal Drop 暖光室内 → 字幕 "New Arrivals"
镜头3 (4s): Classic Hoop 极美人像 → 字幕 "Shop Now"
镜头4 (4s): Shell Pendant 海边阳光 → Logo + CTA

音乐: 轻柔弦乐或 ambient
色调: 全片暖色调统一
字体: 衬线体，白色居中，底部安全区
```

### 各产品 PDP Banner (12s 每条)
```
每个产品独立 banner：
镜头1 (3s): 产品氛围全景
镜头2 (3s): 模特佩戴中景
镜头3 (3s): 首饰特写
镜头4 (3s): 留白 + "Shop Now" CTA
```

### 统一风格约束
- 所有视频保持暖色调（除 Silver Bar 冷调）
- 不要加文字进 AI 视频，后期用剪映加
- 每段视频 4-5s
- 运动强度低，避免首饰变形
